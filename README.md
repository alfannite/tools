# 🛠️ Tools

> Koleksi alat bantu (utility scripts) untuk mempercepat alur kerja sehari-hari.
---

## 📖 Tentang

**Tools** adalah repositori pusat untuk menyimpan berbagai skrip dan utilitas ringan yang saya gunakan dalam pengembangan sehari-hari. Mulai dari otomatisasi folder, konversi format, hingga integrasi API sederhana — semuanya ada di sini.

Repositori ini dirancang agar **modular**, **portabel**, dan **mudah dikembangkan**.

---

## ⚡ Fitur Utama

- ✅ **Otomatisasi Tugas**: Hapus file duplikat, rename batch, dsb.
- 🔄 **Konversi Data**: Ubah format JSON ⇄ CSV, XML ⇄ YAML.
- 📁 **Manajemen Folder**: Buat struktur proyek otomatis.
- 🧪 **Testing Sederhana**: Skrip untuk generate data dummy.
- 🔌 **Extensible**: Tambahkan skrip baru tanpa konflik.
